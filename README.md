# Automatic License Plate Recognition (ALPR)

A Python project that automatically detects and recognizes vehicle license plates from images or video streams using **YOLOv8** for object detection and **EasyOCR** for text recognition.

---

## Features

- Detects license plates in images or real-time video streams.
- Recognizes license plate text using EasyOCR.
- Supports multiple vehicles in a single frame.
- Fast and accurate thanks to YOLOv8.
- Optional logging of recognized plates.

---

## Tech Stack

- Python 3.x
- [YOLOv8](https://ultralytics.com/) (Ultralytics) for object detection
- [EasyOCR](https://github.com/JaidedAI/EasyOCR) for text recognition
- OpenCV for video/image processing
- NumPy
